Latest Nora Updates

Unified /setup Control Hub

We consolidated settings subcommands into a single interactive `/setup` panel. Premium configurations are now accessed securely in an owner-only section using modal popups.

  • Security Enhancements: Consolidated command footprint, preventing parameter exposure in Discord logs.
  • iOS-like Liquid Glass Theme: Support for ambient flowing gradient backdrops and deep glassmorphic blurs.
  • Parallax Team Roster: High-end 3D animated profile cards displaying live Discord status and creation dates.
  • Secure Routing: Full absolute and relative routing fixes ensuring that configurations made on vaztinix.dev sync live.

Consolidated Mod & Verification

We simplified moderation workflows and Roblox setups by consolidating legacy commands into clean, subcommand-driven structures.

  • Unified Warnings (/warn): Replaced multiple standalone commands with a single command supporting add, list, edit, delete, and clear subcommands.
  • One-Click Verification (/verify): Merged Roblox verification steps into a single user-facing slash command for instant account linking.
  • Real-time Cache Invalidation: Implemented active settings cache invalidation, forcing the bot to load new configurations instantly from the database when changes are made via the Nora Studio dashboard or Discord.

Aura Neural Engine

Nora now supports dynamic switching between three intelligent engines: Gemini 1.5 Flash, GPT-4o Mini, and our Local Privacy Core (Qwen 1.8B) which runs 100% on bare metal.
