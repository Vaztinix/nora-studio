Privacy Policy

1. Privacy-First Philosophy

Nora was founded on a strict Zero-Access architecture. We believe that your server's data is private. Developers have no backend overrides to read server chats, configuration logs, or private user details.

2. Data We Collect

To provide baseline functionality, Nora stores limited configuration data in our local SQLite database. This includes guild configurations, level XP scores, custom roles, log channels, issued user warnings/infraction logs, and Roblox verification/link details. We do not store message history permanently.

3. Message Processing

Message content is processed in real-time memory strictly to enforce active moderation rules (AutoMod) or to supply context to chosen AI engines. This processing occurs in memory and is instantly discarded; it is never written to disk or logged.

4. AI Data Policies

We do not share your server's logs, member conversations, or leveling data with third parties to train machine learning models. Interactions with our AI features are sent only to designated API endpoints (OpenAI / Google Gemini) for real-time response generation.

5. Web Dashboard & Analytics Privacy

The Nora Studio Dashboard connects directly to your authorized Discord account to load the servers you manage. We do not permanently store your OAuth2 tokens; they are maintained in session memory and cleared upon logout. Configuration changes made via the dashboard, including Guild Configurations, Integrations, and Top.gg Analytics, are transmitted securely and validated on our backend, ensuring that only users with Administrator or Manage Server permissions can view or edit settings.

6. User Sovereignty & The Kill Switch

We respect the right to be forgotten. Users can run the `/mycard` command inside Discord at any time to inspect their database record, and press the "Erase Data" button to instantly and permanently wipe their leveling statistics, XP, Roblox connections, and warning records from the database across all Nora nodes.

7. Server Data Isolation

Every server is treated as an isolated container. Data, logs, settings, and XP systems generated in one guild are strictly isolated and are never shared or made visible to other servers.

8. Data Erasure upon Bot Removal

If Nora is kicked or removed from a guild, all stored configuration records and log parameters associated with that server ID are scheduled for deletion to keep our databases clean and protect your privacy.
